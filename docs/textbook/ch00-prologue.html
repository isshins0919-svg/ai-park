この冒険で学ぶのは

Claude Codeです。「質問に答えてもらう」のではなく、「目的を伝えたら自分で考えて実行してくれる」AIエージェント。Slack投稿、データ分析、レポート生成、画像作成…全部自分でやる。それがClaude Codeの世界。

💬 Claude Chat の場合

人間「この商品のタイトル案を3つ考えて」
AI → 3案を出す
人間「2番目をもう少し短くして」
AI → 修正版を出す
人間「じゃあこれをセラセンに貼って…」

1往復ごとに人間が指示し、結果をコピペ。

⚓ Claude Code の場合

人間「この商品のAmazon出品ページを最適化して」

① 競合10商品を自動分析
② 検索KWトレンドを調査

「目的」を伝えれば、途中のステップを自分で判断。

WHY NOW

なぜ今こんなに話題なのか

AIは何年も前からあります。でも2025-2026年に起きた3つの変化で、「使える人」と「使えない人」の差が一気に広がりました。

🚀 2025-2026に起きた3つの変化

量の変化ではなく、質の変化

1

脳の容量が100倍に（コンテキストウィンドウ）

2023年: 8,000トークン（文庫本10ページ分）
2026年: 100万トークン（文庫本1,000ページ分）
商品データ、競合レポート、過去の売上…を一度に全部読み込めるようになった。

2

外部ツールに自分で繋がる（MCP）

AIがSlack、Googleドライブ、データベース、ブラウザに自分でアクセスできるようになった。「データをコピペしてAIに渡す」作業が不要に。

3

自分で考えて動く（エージェント実行）

途中でエラーが出たら自分で修正。複数のツールを組み合わせて複雑なタスクを完遂する。「目的を伝えたら勝手にやってくれる」が現実になった。

・競合分析: 毎朝自動で取得・通知
・クリエイティブ: 日50パターンをABテスト
・データ分析: 「分析して」の一言で完了

人間は「判断」と「クライアント対応」に集中。

Amazon × Claude Code の実例

1

Amazon返品処理を2文の指示で自律実行

あるユーザーがX（Twitter）で報告: Claude Codeに「靴のサイズが合わなかったから返品して、次のサイズを注文して」と2文で指示。AIがAmazonにアクセスして返品処理→再注文まで完全に自律で完了。

2

Amazon Ads × MCP連携でPPC自動最適化

SellerMetrics社がAmazon広告APIとClaude CodeをMCPで接続するガイドを公開。広告データの自動取得→分析→入札調整の提案まで、人間はダッシュボードを見るだけ。

3

セッション横断で商品戦略を記憶

Seller Labs社の記事: Claude Codeの最大の強みは「記憶」。商品カタログ、価格戦略、ブランドガイドラインをセッション横断で覚えているから、毎回説明し直す必要がない。

4

受講者の87%が1週間で時間短縮を実感

Claude Code for Business（海外の研修プログラム）の受講者データ: 87%が受講1週間以内に業務時間の短縮を実感。平均で週10-15時間の削減。

📖 この島の航海用語

Claude Chat

Webやアプリで使うClaudeの基本版。質問に答える「相談相手」。ファイル操作はできない。

Claude Cowork

デスクトップアプリ版Claude。ファイルの読み書きや資料整理ができる「事務アシスタント」。2026年4月に正式リリース。

Claude Code

ターミナルやIDEで動くClaude。自分で考え、ツールを使い、コードを書いて実行する「自律エージェント」。この冒険の主役。

AIエージェントえーあいえーじぇんと

「目的を伝えたら自分で考えて実行するAI」の総称。チャットAI（質問に答えるだけ）とは根本的に違う。

MCPえむしーぴー

Model Context Protocol。AIが外部ツール（Slack、Google Drive、DB等）に自分でアクセスするための規格。「AIのためのUSB」。

コンテキストウィンドウCW

AIが一度に処理できる情報量の上限。人間の「ワーキングメモリ」に相当。詰め込みすぎると品質が落ちる。

トークンとーくん

AIがテキストを処理する最小単位。日本語1文字 = 約1〜3トークン。CWの容量はトークン数で測る。

プロンプトぷろんぷと

AIへの指示文のこと。良いプロンプト = AIの出力品質が高い。Island 02で詳しく学ぶ。

🗺️ AIに任せたい仕事TOP3

冒険を始める前に、自分の業務で「AIに任せたい仕事」を3つ書き出す準備運動。
この3つが、Island 10の最終ミッションで実際にAIに任せる対象になります。

ゴール

自分の業務で「AIに任せたい仕事TOP3」を書き出す

  1. 紙やメモアプリに、日々の業務を10個書き出す（5分）
  2. その中から「繰り返しが多い」「時間がかかる」「判断が必要」の3つの基準で優先順位をつける
  3. TOP3を選んで、それぞれ「今何時間かかっているか」を書く
  4. 「AIに任せたら何分になりそうか」を想像して書く（正解はなくてOK）
所要時間: 約10分
成果物

「AIに任せたい仕事TOP3」リスト。この冒険の全ミッションを通じて、これらを実際にAIに任せる設計を作っていきます。
