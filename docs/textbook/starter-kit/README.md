# Starter Kit — クラウドコード ミニダイブ

研修で配布するファイル一覧です。

## そのまま使えるスキル（.claude/commands/ にコピー）

| ファイル | 何ができるか |
|---------|------------|
| `handoff.md` | セッション引き継ぎ。プランと実行を分離する |
| `nice-dive.md` | セッションの学びをログに記録する |

## ルール（.claude/rules/ にコピー）

| ファイル | 何を守るか |
|---------|----------|
| `security.md` | APIキー漏洩、危険な操作の防止 |
| `anonymize.md` | 個人情報の匿名化ルール |

## テンプレート（構造を参考にして自分用に書き換える）

| ファイル | 何の設計図か |
|---------|------------|
| `CLAUDE-template.md` | CLAUDE.md（AIの全体設計図）のテンプレート |
| `identity-template.md` | identity.md（AIの人格設計）のテンプレート |
| `skill-template.md` | スキルファイルのテンプレート |
| `agent-design-principles.md` | エージェント設計の7原則 |

## 使い方

1. `.claude/commands/` に `handoff.md` と `nice-dive.md` をコピー
2. `.claude/rules/` に `security.md` と `anonymize.md` をコピー
3. `CLAUDE-template.md` を参考に、自分の `CLAUDE.md` を書く
4. `identity-template.md` を参考に、自分の `identity.md` を書く
5. 新しいスキルを作るときは `skill-template.md` を参考にする
