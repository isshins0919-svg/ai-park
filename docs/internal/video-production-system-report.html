1 エグゼクティブサマリー

当チームの動画制作は、AI駆動の統合パイプラインにより、「編集依頼（台本CSV）」から「完成動画（mp4）」までをほぼ全自動で完結させています。

  • 10種のAIエージェントが台本の品質を多角的に評価
  • 5つの外部AI（Claude / Gemini / Fish Audio / Grok / Nano Banana）を組み合わせた制作パイプライン
  • 品質ゲート制度により、基準未達の動画は自動ブロック
  • 人間の作業は台本ドラフト作成とテーマ決定のみ（全体の約10%）

DPro連携（外部広告データベース）

  • 同ジャンルのHIT広告TOP5-10を自動取得
  • 再生数・いいね数・推定広告費から「勝ちパターン」を逆算
  • シーン構成・接続詞・フック型・CTA型をDNA分析して抽象化
  • データドリブンで「何が効くか」を学習し続ける仕組み

品質判定の学習基準（ジャッジシステム）

total_score =
    フック x 25% + 感情アーク x 20% + CTA x 20% + ブリッジ x 15%
  + テンポ x 10% + スタイル x 5% + マッチ x 5%

total_score >= 80

12 まとめ

当チームの動画制作システムは、「AIが品質を担保し、人間は創造的判断に集中する」という設計思想のもとに構築されています。

現状
  • 台本から完成動画まで、人間の作業は全体の約10%
  • 10種のAIクルーによる多角的品質評価
  • ブリッジスコアによる動画→LP一貫性の担保
今後
  • 実績データのフィードバックループでAIが自ら精度向上
  • 素材生成AIの進化で素材ゼロからの完全自動制作が現実的に
  • 最終的には企画→制作→配信→分析→改善の全サイクルをAI主導で回す未来
