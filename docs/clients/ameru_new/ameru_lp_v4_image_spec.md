# ameru LP v4 — 画像仕様書（撮影指示書）

**対象LP:** `reports/ameru_lp_v4_production.html`
**作成日:** 2026-04-18
**目的:** このドキュメントに従って画像を撮影・納品すれば、LPに差し替えるだけで本番投入できる状態に仕上げる

---

## 🎯 撮影全体の基本方針

| 項目 | 指定 |
|---|---|
| **光源** | 原則すべて**自然光**（北向き窓の柔らかい光が理想）。ストロボ・リングライトは使わない |
| **背景** | 淡水色の画用紙（`#E8F5F8` 相当）／ 生成りのリネン布／ 白の木目／ 白壁 の4種を使い分け |
| **色温度** | 5200〜5500K（やや暖かめ）。ホワイトバランスは白壁基準で固定 |
| **トーン** | 彩度 ▲10% / コントラスト ▲10% で「ふんわり」感を作る。後処理で統一 |
| **被写体カラー** | ずもっぴのスカイブルー（`#8FCFDD` 付近）がブランド色。撮影時にかぶりを確認 |
| **小物** | ヴィンテージのカップ／木製トレイ／リネン／乾いた花 など「生活感×上品」 |
| **NG** | 派手な造花／テカリ反射／ストロボの硬い影／蛍光灯カラー |
| **ファイル形式** | JPG 80%品質（ヒーロー画像のみPNG書き出しも可） |
| **カラープロファイル** | sRGB |
| **命名規則** | `ameru_<SLOT_ID小文字>.jpg`（例：`ameru_fv_hero.jpg`） |

## 🎨 プレースホルダの差し替え方法

### 現状（差し替え前）
```html
<figure class="img-slot">
  <span class="img-slot__id">FV_HERO</span>
  <span class="img-slot__title">完成したずもっぴ<br>手のひらに載せた俯瞰カット</span>
  <span class="img-slot__meta">正方形 1:1 ／ 自然光／背景は白〜淡水色</span>
</figure>
```

### 差し替え後（本番）
```html
<img src="images/ameru_fv_hero.jpg"
     alt="完成したずもっぴを手のひらに載せた俯瞰カット"
     width="480" height="480"
     loading="eager"
     style="width:100%;display:block;border-radius:50%;">
```

**重要な注意:**
- FV_HEROだけは `loading="eager"`（lazy禁止）← EC-Forceルール遵守
- その他はすべて `loading="lazy"`
- `alt` は写真の内容を日本語で正確に
- `width` / `height` 属性を入れる（CLS=レイアウトシフト防止）

## 🎬 撮影時のビジュアル統一ルール

### 光
- 窓から45°の斜光が基本
- レフ板で影を起こす（コントラストきつくしすぎない）
- 時間帯: 午前10時〜午後3時の柔らかい光

### フレーミング
- 主役は画面中央〜やや右下に配置
- 上下左右に**必ず余白**を残す（後でクロップ調整できるように）
- 余白は画面の **25%以上**

### 編集
- Lightroom プリセット（後日作成）で全カット一括調整
- トーンカーブ: シャドウ +10 / ハイライト -10 / 彩度 +5
- 白飛び・黒つぶれNG

## 🚨 撮影で絶対にやってはいけないこと（地雷リスト）

1. **完成品のずもっぴを1枚もない状態でLP公開** — 現状LPの最大の問題点
2. **パッケージ単体ばかり撮る** — 前バージョンでパッケージ画像を4回使い回していた。今回は変化をつける
3. **ストロボ撮影** — ameruの世界観は自然光でしか作れない
4. **やらせ感のあるUGC** — VOICE_PHOTOはあえてプロ感を消す
5. **No.2〜5を空の円で表示** — 不安を増すので必ずシルエット/モックを入れる
