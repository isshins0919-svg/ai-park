## ② トップ5選定 + 深掘り

ameruの現状（スカイブルー×白／パッケージ主役／EC-Force制約／Love is+ずもっぴIP／N1=推し活ワーカー）に照らし、**今すぐ移植可能で、かつ本質的に学びが大きい5本**を深掘りします。

### Top 1: Craftie Home Box（https://home.craftie.jp/）

**なぜameruの参考になるか**
最も近い競合であり、かつ「手芸×サブスク×初心者×可愛い」の四要素を日本市場で成立させている唯一に近い存在。ameruが目指すべき「暮らしに溶けるビジュアル」の実在解。

**構図/配色/余白/文字量**
- オフホワイト＋ナチュラル木目、差し色はキット作品そのものの色
- FVは完成作品の俯瞰カット1点主役、コピーは「毎月届くハンドメイド体験」の一文
- 以降は作品カード＋暮らしカット＋BOX中身の3種類を循環
- 文字量は全体で ameru 現状LPの体感 1/2〜1/3

**ameruに移植可能な具体要素**
- FVを「完成したずもっぴ1体の俯瞰」に置き換え、コピーを「編み始め完成済み。毎月1体、あなたのずもっぴ」レベルまで絞る
- 「パッケージ主役→完成品主役」への重心移動（これが一番効く）
- 月次BOX開封カット（袋を開ける瞬間の手元）を追加
- 文字を削り、写真の余白を1.5倍に

**移植できないポイント/注意点**
- Craftieは「毎月違う作品」だが、ameruは「ずもっぴIP世界観の連作」。作品の並列見せをそのまま真似すると世界観が薄まる
- ブランドカラーが違う（Craftieはベージュ系、ameruはスカイブルー）ので色のトーンだけ移植、構成を真似る

## ③ 本質的な変化提案: ameru LP再設計

### 1. 現状LPの弱点診断（推定）

「可愛いビジュアルで買いたい」に届かない理由は、おそらく以下の構造にある:

1. **パッケージ主役なのに、パッケージそのものが主役級の美しさに届いていない**
   FUJIMI・SHIROレベルだと、パッケージ1枚で世界観が完結する。ameruのパッケージ写真は「商品紹介写真」に留まっている可能性。
2. **FVに情報を詰めすぎている**
   SHIRO・snaq.meは「FVで売らない」。ameruは6セクション構成の1枚目で勝負しようとして、結果ぼやけている懸念。
3. **「ずもっぴ」「Love is...」「スカイブルー」という3つのIPが並列していて、主役が決まっていない**
   メイン主役→サブの関係を明確化していない可能性。
4. **「完成品（あみぐるみのずもっぴ）」が主役になっていない**
   Craftie・The Woobles・snaq.meはすべて「完成体験」を主役に。ameruは「キット」を主役にしているのでは。**キットではなく、キットで作れるずもっぴ（完成品）を主役にする**べき。
5. **「世界観カット」が不足**
   編んでいる手元／完成したずもっぴの日常／部屋に飾られたずもっぴ、など世界観を広げるカットが弱い可能性。

### 2. 構造レベルの変更

**現状: FV / パッケージ / キャラ / コレクション / How / Offer（6セクション）**

**提案（8セクション／長尺化・密度再配分）:**
