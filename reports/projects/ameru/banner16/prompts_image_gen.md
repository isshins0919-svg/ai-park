## 共通 STYLE（全バナーに前置）

```
【フォーマット厳守】
- Instagram / Meta feed用 スクエア 1:1（1024x1024）
- 日本のハイエンドD2Cサブスク（SHIRO / FUJIMI / Craftie Home / snaq.me）レベルの仕上がり
- ターゲット: らぶいーず®（日本テレビ公式）ファン／手芸に関心のある20-40代女性

【ブランドカラー（厳守）】
- ベース: ピンクベージュ (#F4E5DC) / アイボリー (#FBF5EC) / ミルクホワイト
- キャラアクセント:
  * すもっぴ = 水色×白のクマ
  * ぴょんちー = ピンクのうさぎ
  * にゃぽ = クリーム色の猫＋黄色リボン
  * うるる = グレー〜薄紫
  * ぱおぱお = ミントグリーンのぞう
- 自然光・柔らかい陰影・余白たっぷり・上品・大人可愛い・クラフト感

【日本語テキストの描画ルール】
- 日本語は絶対に誤字・崩れNG。正しく美しくレンダリング
- 見出しは太め丸ゴシック、サブは細め丸ゴシック
- 文字は大きめ・シンプル・短く。画面に対して余裕ある配置
- 英数字は等幅サンセリフ
- ameruロゴは左下に小さく（小文字 "ameru"）
- らぶいーず®日本テレビ公式バッジは右上に小さく

【絶対NG（禁忌）】
- 英語表記「Love is...」「loveeez」「love-eez」「Love Eez」は絶対NG（らぶいーずは完全ひらがな）
- 「本日より」「今日から」「24時間限定」「いまだけ」等の時間依存ワード
- 「買うより〇〇」「買うのではなく」等のネガ対比
- 「すもっぴ誕生日」「◯月スタート」等の誕生月トリガー
- 「1匹」「1個」（キャラは「1体」統一）
- 競合dis・他社名
- ケバケバしいフリー素材感、ストック写真感、AIフラットイラスト調
- 人物の顔は出さない（手元・肘まで・背後ショットのみOK）
- キャラの色ブレ（水色=水色、ピンク=ピンク、ミントは緑系）
```
