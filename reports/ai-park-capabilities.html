1. identity.md 読み込み → 人格を認識
2. git pull origin main --rebase → 最新状態に同期

「おつかれ」「ばいばい」「おわり」「また明日」「終了」「寝る」
→ git add -A → commit "sync: YYYY-MM-DD" → push origin main

マーケティング哲学サマリー

🎈 最上位目的
「今この瞬間、この美しき世界を愛し、愛されること。」全てのビジネスはこの目的のためにある。
🎯 N1モデル
N1を見つけ、需要のニュアンスを捉え、コンセプトを作って、コンセプトに一貫したコミュニケーションで脳内1番乗りする。
🏆 Only1 > No1
No.1ポジションではなく、Only1のポジションを取る。差別化ではなく、唯一無二であること。
🎨 クリエイティブ3つの魂
①KVそのもの（ブランドKVレベルで美しい）②キラーキャッチコピー（1行で全てを語る）③別の仮説（各クリエが固有の検証仮説を持つ）
🔀 戦略翻訳
クリエイティブは「生成」するものではなく、strategy.jsonという設計図を各フォーマットに「翻訳」するもの。
🛡️ ベクトルは守護神
ベクトルは「良いものを作る魔法」ではなく「ダメなものを通さないフィルター」。品質ゲートとして機能する。

3層認知設計（ターゲット層の分類）

潜在層 → 「原因の気づき」+「解決策の方向性」
準顕在層 → 「既存の限界」+「新しい判断基準」
顕在層 → 「根本的な違い」+「パラメータ書き換え」

プライバシー・ルール

個人情報保護（必須・自動適用）

  • 氏名 → 顧客A / 顧客B / 解約者A … に自動置き換え
  • 居住地 → 都道府県レベルに粗粒化（市区町村以下削除）
  • 年齢 → 「40代」「50代」などに丸める
  • メール・電話 → [非公開] に置き換え
  • 勤務先 → 業種のみ残す（「建設業」「事務職」）
  • HTMLレポート・GitHubにpushするファイルに実名を含めない
  • ユーザーから指示がなくても、AIが先に匿名化してから出力する
