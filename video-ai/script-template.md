# 動画台本テンプレート — カントク君渡し用フォーマット

> このテンプレートで台本を書くと、カントク君がそのまま9君に投げられる。
> 人間の仕事: このファイルを埋める。AIの仕事: 評価・改善・編集指示書化。

---

## STEP 0: 動画メタ情報（必須）

```
商品名         :
商品カテゴリ   : （例: 体臭ケアサプリ / 美容液 / 法律相談）
LP URL         :
ターゲット     : （例: 40代以降で加齢臭が気になり始めた男性）
フック型       : A / B / C / D（わからなければ「未定」でOK）
LPの冒頭テキスト: （LP上部300文字ほどコピペ）
```

> **フック型の選び方**
> - A（衝撃発見型）: 健康・サプリ・腸活系
> - B（日常会話型）: 美容・スキンケア・デオドラント系
> - C（直撃型）: 法律・金融・BtoB系
> - D（権威×数字型）: 健康食品・ダイエット・医療系

---

## STEP 1: QUALIFY設計（書いてから台本を書く）

台本を書く前に答えること。ここがズレると全体がズレる。

```
絞り込みワード（2〜3個）:
  例: 「40代以降」「毎朝シャワーしても夕方が気になる」「デオドラントが効かない」

LPに残す情報（動画では言わないこと）:
  例: 成分の詳細メカニズム / 1日の摂取量 / 価格の詳細

動画で言ってよいこと:
  例: 「体の内側に原因がある」「外側ケアでは根本解決しない」「6ヶ月で変化を実感した人続出」
```

---

## STEP 2: 30スロット台本

**ルール（必ず守る）**
- 1スロット = 2秒 = **最大12文字**
- STOP（S1-2）: 9〜10の強さで止める
- QUALIFY（S3-10）: 絞り込みワードを入れる。関係ない人は離脱してOK
- BUILD（S11-25）: 答えを全部出さない。「続きはLPに」のギャップを作る
- DRIVE（S26-30）: LP遷移への衝動を最大化。オファーを数字で出す

```
【STOP フェーズ — スクロールを止める】

S01 (0-2s)  :
S02 (2-4s)  :

【QUALIFY フェーズ — 正しい人だけを残す】

S05 (8-10s) :
S06 (10-12s):
S07 (12-14s):
S08 (14-16s):
S09 (16-18s):
S10 (18-20s):

【BUILD フェーズ — 問題×解決を積み上げ、答えはLPに残す】

S11 (20-22s):
S12 (22-24s):
S13 (24-26s):
S14 (26-28s):
S15 (28-30s):
S16 (30-32s):
```

## STEP 3: 各スロットの補足（任意）

台本だけでは伝わらない演出意図があれば記載。

```
S__ : （例: ここに商品パッケージのアップを入れる / ナレーションなし・テロップのみ）
S__ :
```

## 書き方ヒント（よくある失敗）

### STOP（S1-2）でやりがちな失敗
```
NG: 「プルーストで体臭ケア」← 商品名から入る。誰も止まらない
NG: 「体臭でお悩みの方へ」  ← 弱すぎ。驚きがゼロ
OK: 「衝撃！加齢臭放置で何が起きる？」
OK: 「朝のいい匂いどこ行った？」
```

### QUALIFY（S3-10）でやりがちな失敗
```
NG: 「体臭が気になる人は...」← 漠然。誰も「自分のことだ」と思わない
NG: 「誰でも使えます！」    ← 絞り込みゼロ。LP到達後にミスマッチ多発
OK: 「40代以降で急に臭いが変わった経験ない？」
OK: 「毎朝シャワー浴びても夕方が気になる...」
```

### BUILD（S11-25）でやりがちな失敗
```
NG: 「プルーストの成分Xが皮脂の酸化を抑制して...」← 言いすぎ。LPに来る必要がなくなる
OK: 「体の内側からしかケアできない理由がある」
OK: 「なぜデオドラントでは解決しないのか、その答えとは...」
```

### DRIVE（S26-30）でやりがちな失敗
```
NG: 「詳しくはリンクから」← 弱すぎ。何も動かない
NG: 「気になった方はぜひ」← 緊急性ゼロ
OK: 「今なら初回210円。試さないと損」
OK: 「この方法、知らないまま損しないで」
```

## カントク君への渡し方

このファイルを埋めたら：

```
/director
↓ 以下を貼る:
- このファイル（台本 + メタ情報 + QUALIFY設計）
- 素材フォルダのファイル名一覧
```

カントク君が受け取り次第、9君を全部起動します。
