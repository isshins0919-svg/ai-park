## プラン詳細（設計メモ）

### v2パイプラインの核心設計

```
素材構造（ちはる-アポバス）:
  [女性] 「○○と言ってください」  ← プロンプト（使わない）
  [男性] NG1「○○」              ← 不一致テイク（最後じゃないから自動スキップ）

  [男性] OK「○○」               ← find_best_take()が最後の一致区間を選択 ✅
  → 次シーンへ
```

### narration_mode の切り替え方法
- CSVの `ナレーション` フィールドに `なし` と書く → `original` モードに自動切替
- `--keep-audio` フラグでも強制的にoriginalモードに
- originalモードではTTS生成をスキップし、前処理済み音声をそのまま使う

### BGM
- 固定10%（`volumex(0.10)`）
- よく使う音楽素材: `/Users/ca01224/Desktop/動画編集フォルダ/よく使う音楽素材/`

### preprocess_talking_head_v2.py の重要パラメータ
```python
MATCH_THRESHOLD = 0.60   # 日本語口語の表記ゆれに対応
SCENE_PAD_END = 0.3      # テイク末尾に0.3秒余白
LOUDNORM_FILTER = "loudnorm=I=-18:LRA=11:TP=-1.5"  # 声の音量統一
BRIGHTNESS_FILTER = "eq=brightness=0.05:contrast=1.1"  # 明るさ補正
```
