## プラン詳細（設計メモ）

### 現在のフロー（正式）
```
ユーザー: /director <CSV> <素材フォルダ>
  ↓
🎬 カントク君 (director.md / claude-opus-4-6):
  1. Read で CSV読み込み
  2. 台本→30スロット×2秒グリッドに変換
  3. Agent × 8でエージェント並列評価
  4. /quality-judge でGO/REVISE/BLOCK判定
  5. GO → Bash で edit_ai_v2.py 自動実行
  ↓
edit_ai_v2.py:
  --script CSV --clips 素材フォルダ --output output.mp4
  処理: CSVパース → TTS速度推定 → シーン単位TTS → 映像生成 → 結合 → 出力
```

### 台本設計の重要ルール（カントク君が覚えておくこと）
1. **素材なし or 同一素材の連続シーン** = 前のシーンの素材を継続使用（クリップが止まらずに流れる）
2. **固定注釈** = 薬機法テキスト（CSVの「■ 固定注釈」セクション）。空の場合は警告を出す
3. **カット注釈** = 各シーンの根拠URL・補足テキスト。画面上部に小さく表示

### edit_ai_v2.py の重要引数
```bash
python3 video-ai/edit_ai_v2.py \
  --script   <CSV or HTML>      # 台本ファイル
  --clips    <素材フォルダ>      # 動画・画像・BGMが入ったフォルダ
  --output   <出力mp4>           # 出力先
  --voice    <声カテゴリー>      # 省略時はCSVから自動取得
  --ref-video <参考動画>         # テロップスタイル抽出（オプション）
  --no-ai                        # API不使用（テスト用）
```
