# さくら編集ガイド — ASMR特化の音・映像ルール

## 音声処理の鉄則

### 1. ノイズ除去（軽めに）

```bash
ffmpeg -i input.mov -af "afftdn=nf=-25" output.wav
```

- `nf=-25` は軽めの設定（呼吸音・息づかいを残す）
- `nf=-35` は強め（環境音を消すがASMRの「生々しさ」も消える）
- **さくらは-25推奨**（声質の親密感を保つ）

### 2. ラウドネス正規化

```bash
ffmpeg -i input.wav -af "loudnorm=I=-16:TP=-1.5:LRA=11" output.wav
```

- `I=-16` LUFS（TikTokの推奨値）
- `TP=-1.5` True Peak（クリッピング防止）
- `LRA=11` ラウドネスレンジ（囁き↔通常声の幅を保つ）

### 3. 無音区間の扱い — **絶対に削らない**

- 壺ポイント（2-3秒の無音）は**ASMRの命**
- `silencedetect` で検出しても、**削除しない**
- 息づかいだけが聞こえる時間を尊重する

### 4. 音量の目標値

| 部位 | RMS目標 |
|---|---|
| プロ声（張り） | -16 dBFS |
| 素の声（通常） | -18 dBFS |
| 囁き | -22 dBFS |
| 息づかい | -30 dBFS |
| 壺ポイント（無音） | -50 dBFS 以下 |

## 映像処理

### 1. 縦型クロップ

- 解像度: **1080 × 1920**（9:16）
- ソースが横型の場合はセンタークロップ
- iPhone縦撮りの rotation metadata を正しく処理

### 2. カット割りルール

- **3秒以上同じ画角を続けない**（飽き防止）
- ただし**ASMRシーン（Script #4）は例外**（1シーン長めOK）
- シーンの切り替えはフェードではなく**ハードカット**（ASMRの没入感）

### 3. 色調

- 暖色系フィルター +10%
- 彩度 -5%
- 黒を引き締める（シャドウ +10、ブラックレベル -5）

## テロップ設計

### カラールール

| 音声タイプ | 色 | 備考 |
|---|---|---|
| プロ声 | `#FFFFFF` 白 | 半透明黒背景 or ドロップシャドウ |
| 素の声 | `#E89BB0` ピンク | さくらのブランドカラー |
| 選挙プロ | `#E8A855` オレンジ | Script #5用 |
| 無音 | — | テロップなし |

### フォント・配置

- フォント: Hiragino Sans W6 / Noto Sans JP Bold
- サイズ: 76pt
- 配置: 画面下72%位置
- 1行最大: **12文字**（v2の14文字より短く）
- 行数: **1行のみ**（ASMRは情報過多NG）
- 背景: 半透明黒 `rgba(0,0,0,0.5)`、角丸10px
- ドロップシャドウ: 2px、ブラー4px

### 改行処理

- BudouXで自動改行
- 禁則処理（句読点・カッコ）
- 「...」は改行しない

## シーンごとのテロップ方針

### プロ声シーン
```
「1番、センターフィールダー、背番号51——」
↓
白テロップ: 「1番、センター」→ 「背番号51」
```

### 素の声シーン
```
「...4番の名前呼ぶとき、声、変わっちゃうの」
↓
ピンクテロップ: 「4番の名前呼ぶとき」→ 「声、変わっちゃうの」
```

### 無音シーン
- テロップなし
- 画面下に小さく「...」だけ表示してもOK（演出）

---

## BGM / 効果音方針

### BGM
- **入れない**（ASMR優先）
- 例外: Script #5（選挙）のみ、オチ部分に微音で効果音OK

### 効果音
- 基本なし
- 台本ページをめくる音は素材音声のまま活かす
- マイクの「ブツッ」音は除去

---

## 品質チェックリスト

編集完了後、以下を必ず確認:

- [ ] 総尺 50〜65秒
- [ ] ラウドネス -16 LUFS ± 1
- [ ] ピーク -1.5 dB以下
- [ ] テロップ誤字なし
- [ ] テロップ色分けができている
- [ ] 壺ポイント（無音）が保たれている
- [ ] カット割りが3秒以上続いてない（#4除く）
- [ ] 顔が映り込んでない（首下 or 後ろ姿のみ）
- [ ] 個人特定情報なし（名札・住所・ID等）
- [ ] 音声が左右片方だけになってない（モノラル or センター）

## FFmpegコマンド集

### 音声前処理（フルセット）

```bash
ffmpeg -i input.mov \
  -af "afftdn=nf=-25,loudnorm=I=-16:TP=-1.5:LRA=11" \
  -c:v copy \
  output.mov
```

### 縦型クロップ（センター）

```bash
ffmpeg -i input.mov \
  -vf "crop=ih*9/16:ih,scale=1080:1920" \
  -c:a copy \
  output.mov
```

### 縦型動画のrotation正規化

```bash
ffmpeg -i input.mov \
  -vf "transpose=1" \
  -metadata:s:v:0 rotate=0 \
  -c:a copy \
  output.mov
```

### セグメントカット

```bash
ffmpeg -ss 0.5 -i input.mov -t 4.0 -c copy output.mov
```

### 音量レベル測定

```bash
ffmpeg -i input.mov -af "volumedetect" -f null - 2>&1 | grep "mean_volume"
```
