## 1. 音楽が果たす役割

### プラットフォームの構造
- TikTokユーザーの **88%** が音声オンで視聴
- Instagram Reelsも **80%以上** が音声オンで視聴
- 上位パフォーマンス動画の **93%** が音声を一次的ストーリーテリング要素として使用

### 音楽の定量効果
| 効果 | 数値 |
|---|---|
| 音楽あり動画のエンゲージメント率向上 | +15% |
| トレンドサウンド使用時のエンゲージメント向上 | +66〜120% |
| ソニックブランディングによるブランド想起率向上 | 8.5倍（視覚のみ比） |

### 音楽の機能的役割（4つ）
1. **注意喚起** — 冒頭0〜3秒でスクロールを止める
2. **感情誘導** — 世界観・商品イメージを聴覚で先取りさせる
3. **記憶定着** — 視覚+聴覚の同期により想起率が高まる
4. **行動促進** — テンポと音圧でCTA前の行動意欲を高める

## 3. フェーズ別 音楽の役割（STOP/QUALIFY/BUILD/DRIVE）

### STOP（0〜3秒）— スクロールを止める
- 音楽は**フレーム1枚目から**流す。無音スタートは離脱リスク急上昇
- ビートの「1拍目」か「ドロップ直前」を冒頭に持ってくる
- サビ・コーラス・ドロップ部分を先頭に置く（イントロをそのまま使わない）
- 視聴者の45%は最初の3秒を見れば30秒以上視聴継続

### QUALIFY（3〜8秒）— 「自分ごと化」させる
- BGMを最も下げるフェーズ（VO優先）
- ビートは維持しつつ、音量だけコントロール
- メロディラインは単純化されたものが望ましい

### BUILD（8〜25秒）— 証拠・説明で信頼を積む
- ナレーションの抑揚に合わせて音量を微調整
- 「問題→解決」の転換点でBGMの展開（コードチェンジ）が入ると効果的
- ビフォー部分：やや重め → アフター部分：明るく展開

### DRIVE（25秒〜終了）— CTA・購入行動へ
- 音楽を最も「前に出す」フェーズ
- テンポ感を取り戻し、エネルギーを高める
- CTA表示のタイミングでドラムヒットや音の強調を入れる

## 4. 音量ミックスの鉄則（dB設定）

### ナレーション（VO）あり

| トラック | 推奨レベル（dBFS） |
|---|---|
| ナレーション（VO） | ピーク -6〜-3 dB |
| BGM（通常区間 = QUALIFY/BUILD） | -18〜-24 dB |
| BGM（HOOK/CTA区間 = STOP/DRIVE） | -12〜-15 dB |
| 効果音（SE） | -12〜-9 dB |

**実務ルール:**
- VO比でBGMを **-15〜-20 dB 低く** 設定するのが基本
- 絶対に0 dBを超えない

### ナレーションなし（テロップ+音楽のみ）
- BGM: **-10〜-14 dBFS**（主役として前に出す）

## 5. フェードイン・アウトの設計

### フェードイン
| ケース | 秒数 |
|---|---|
| HOOK重視（音楽を強く聴かせたい） | フェードなし or 0.1〜0.3秒 |
| BGMを徐々に立ち上げる | 0.5〜1.5秒 |
| ナレーションから始まる | 1〜2秒 |

### フェードアウト
| 動画尺 | 秒数 | タイミング |
|---|---|---|
| 15秒 | 0.5〜1秒 | 最後のCTAテロップと同時 |
| 30秒 | 1〜2秒 | 終了の1〜2秒前から |
| 60秒 | 2〜3秒 | 終了の2〜3秒前から |

**鉄則:**
- フェードアウト開始はビートの区切り（小節の頭）に合わせる
- CTAが表示されている間は音楽を消さない

## 6. 音楽選定フロー（決定木）

```
STEP 1: 商材カテゴリ → BPM帯とジャンルを決定（上記マップ参照）
    ↓
STEP 2: フック型を確認
  ├── 問題提起型  → やや重め・マイナーベース → BUILDでメジャーに転換
  ├── 感情共感型  → 温かみあるAcoustic / Piano
  └── インパクト型 → ドラムヒット入り / テンポ速め
    ↓
STEP 3: ターゲット層
  ├── 女性30〜50代 → BPM 85〜105 / Acoustic / Lo-Fi
  ├── 女性10〜20代 → BPM 105〜125 / TikTok Trending Pop
  └── 男性全般    → BPM 100〜120 / Electronic / Light Hip-hop
    ↓
STEP 4: 動画尺
  ├── 15秒 → BPM120+、サビ・ドロップから開始
  ├── 30秒 → 標準BPM（100〜120）、感情アーク設計可
  └── 60秒 → BPM広め、中間で展開変化を入れる
    ↓
STEP 5: 音源ソース選択（下記参照）
```

## 7. やってはいけないこと（アンチパターン）

| NG | 問題 | 正しい対処 |
|---|---|---|
| イントロをそのまま冒頭に使う | HOOKを殺す | サビ・ドロップから開始 |
| ナレーションと音楽が同音量 | VOが聞こえない | BGMは-15〜-20 dB下げる |
| ボーカル入り曲をVO広告に使う | 音声が二重で混乱 | 必ずインストゥルメンタルを選ぶ |
| 全編同じ音量で流す | メリハリなく飽きられる | フェーズ別に音量変化させる |
| 無音スタート | 最初0.3秒で離脱率急上昇 | フレーム1枚目から音楽を流す |
| 音楽が突然切れる | 不快な後味 | 必ずフェードアウトを入れる |
| 著作権確認なしで有名曲を使用 | DMCA・広告停止リスク | Commercial Libraryに限定 |

### 音楽が不要・逆効果になるケース
- B2B専門的ウェビナー告知（音楽が軽さを演出し信頼を損なう）
- 強烈ビフォーアフター（肌荒れ・体型）（無音の方がリアル感が出る場合）
- ユーザーの生声UGC（本物感を壊す）

## 10. 2パターン対応の設計方針

音楽ソースには**2つのパターン**があり、クルーは両方に対応する:

| パターン | 概要 | クルーの動き |
|---|---|---|
| **A: 自動選定** | DPro分析 × 手元ライブラリから最適曲を選ぶ | DPro→傾向抽出→ライブラリマッチング |
| **B: 依頼者指定** | 編集依頼時に「このMP3を使って」と指定される | 指定ファイルをそのまま挿入。音量設定のみ自動最適化 |

**B指定の場合も音量ミックスは自動最適化する（VO比での dB 調整は必ず行う）。**
