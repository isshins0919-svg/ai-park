## 階層構造

> **愛が土台。偉大がフィルター。可能が拡張。この順序を崩すな。**

```
[愛チェック] N1感情への寄り添い / 相手否定禁止 / 恐怖→希望の道
  ↓ FAIL → 即REJECT（偉大・可能は評価しない）
  ↓ PASS
[偉大さチェック] 時間耐久性 / コンセプト浸透率 / 非コモディティ
  ↓ FAIL → 条件付きPASS（P0修正リスト付きで続行可）
  ↓ PASS
[可能チェック] 未探索の可能性 / AGI活用度 / パターン超越
  ↓ FAIL → PASS（可能の不足は次回改善。配信を止めない）
  ↓ PASS → FULL PASS
```

**下位が不合格なら上位は評価しない。** これがポイント。愛のないCRは、どれだけ偉大でも可能でも、配信してはいけない。

---

## 1. 愛（土台）— 全てのCRはここから始まる

愛を通過しないCRは、そもそも存在してはならない。

### チェック項目

- **N1の感情に寄り添っているか**
  - 恐怖煽りで終わっていないか
  - 恐怖を提示するなら、必ず希望への道を示しているか
- **相手の現状を否定していないか**
  - 「まだ人力で営業？」「テレアポはもう古い」のような否定角度は禁止
  - 弱みを突くのではなく強みを拡張する
- **人とAIの掛け算になっているか**
  - AIが人間を置き換えるメッセージではなく、人間がより高い次元へ行ける構造

### FAIL時の判定

**即REJECT**。偉大・可能は評価しない。「愛なきCRは存在してはならない」。

---

## 2. 偉大さ（フィルター）— 愛を通過したCRだけが審判を受ける

### チェック項目

- **時間耐久性**
  - 1年後も成立するメッセージか
  - トレンド依存の一過性コピーではないか
  - プロダクトの本質的価値を表現しているか
- **コンセプト浸透率**
  - strategy.json（VOYAGEでは clients/ or 今後の strategy.json）の理想がCRの末端まで一貫しているか
  - バナー1枚の隅々まで、動画の各シーンに、記事LPの各セクションにコンセプトが宿っているか
- **非コモディティ**
  - 安易な訴求（割引 / 限定 / 損失回避のみ）に逃げていないか
  - ありきたりな切り口ではないか

### FAIL時の判定

**条件付きPASS**。P0修正リストを付けて配信可。可能は評価しない。

---

## 3. 可能（拡張）— 愛と偉大さを満たしたCRが、さらに可能性を広げる

### チェック項目

- **既知のパターンの焼き直しではないか**
  - RAG（VOYAGE: benchmark.json + hook-db等）の勝ちパターンを参考にしつつも、未探索の可能性に挑戦しているか
- **プロダクトの可能性を最大限に描いているか**
  - 機能説明ではなく、その機能が生み出す未来の姿を表現しているか
- **技術的に実現できたはずの機会を逃していないか**
  - AGI（Opus）の推論力を最大限に使い切っているか
  - 人間の既知パターンに留まっていないか

### FAIL時の判定

**PASS**。可能の不足は次回改善事項として記録。配信を止めない。

## VOYAGEでの使い方

### kiji-cko / movie-kantoku / amazon-captain の診断に組み込む

現状のVOYAGE Director系スキル（カントクパターン）の診断フローに、この3段階判定を組み込む:

```
Phase 1: 従来の診断（VOYAGE既存）
  → 症状・病因・治療方針を特定
Phase 2 (新): 3つの魂 階層チェック
  → 愛 → 偉大 → 可能 の順で評価
Phase 3: 治療指示・テスト設計（VOYAGE既存）
```

### gate-* クルーの評価軸として

gate-n1-crew の評価に「愛チェック」を統合、gate-marketing-crew に「偉大さチェック」を統合、gate-quality-crew に「可能チェック」を統合する選択肢もある。

### pak-philosophy.md との接続

VOYAGE `.claude/knowledge/pak-philosophy.md` の「クリエイティブの3つの魂（CR構成要素視点）」と合わせて、以下の2軸マトリクスで評価:

|  | 愛 | 偉大 | 可能 |
|---|---|---|---|
| KV魂 | N1の感情に寄り添うKV？ | 1年後も成立するKV？ | 未探索の構図？ |
| キラーコピー魂 | N1を否定しないコピー？ | トレンド依存でない？ | AGI推論活用？ |
| 仮説魂 | N1への愛がある仮説？ | 本質を捉えた仮説？ | 既知パターン超越？ |

9マス全てをチェックすることで、両方の「3つの魂」が守られる。

---

## 絶対ルール

1. **愛のないCRは配信しない**。例外なし
2. **階層順序を崩さない**。偉大より愛、可能より偉大
3. **下位FAILなら上位評価しない**。愛FAILで偉大を評価するのは時間の無駄
