## 0. 久野将司氏の根本哲学（最重要）

**「優先順位の誤りが、売上が伸びない本当の理由」**

多くのセラーが犯す失敗:
- SEOや広告に注力するが、土台（転売対策・レビュー・セール）が整っていない
- レビューが整わないうちに広告を出しても、CTR・CVR共に低く採算が取れない
- セール設計を間違えてクーポンを乱用し、販売実績価格を汚染してしまう

**正しい優先順位: STEP1転売対策 → STEP2レビュー → STEP3セール → STEP4CTR → STEP5CVR → STEP6SEO → STEP7広告**

## 1. Amazonランキングを決める3大要素（久野将司氏の定義）

| 要素 | 重要度 | 説明 |
|------|--------|------|
| **販売実績** | ★★★★★（最重要）| 検索KWごとの販売数が順位を決める。テキストSEOはあくまで「前提条件」 |
| **関連性** | ★★★★☆ | タイトル・説明文にKWを自然に含める。これだけではランキングは上がらない |
| **顧客体験** | ★★★☆☆ | 在庫切れゼロ・FBA利用・レビュー評価。検索順位の「安定化」に寄与 |

**核心原則**: AmazonはAmazonが儲かる商品を上位表示する。
= 「CVRが高い × 販売速度が速い = Amazonが儲かる」 = ランキング上昇。

## 2. Camicksの転換率設計（月商1,000万円からの逆算）

```
月商目標: ¥10,000,000
÷ 販売単価: ¥2,480
= 必要販売数: 4,032足/月

÷ 転換率目標: 15%
= 必要セッション数: 26,880/月

÷ 30日
= 必要日次セッション: 896/日（≈ 900/日）

つまり: 日次900セッション × CVR15% = 134足/日 → 月4,032足達成
```

**CVR15%達成の3要素:**
1. **メイン画像**（検索CTRを決める）: 白背景 + 商品85%以上 + 高コントラスト
2. **サブ画像アーク**（ページ内CVRを決める）: 機能→根拠→価格正当化→感情の流れ
3. **価格 × レビュー数**（信頼を決める）: ¥2,480は高額帯 → 根拠画像必須

## STEP1: 転売対策

**目的**: カートを守る × 転売業者による価格汚染を防ぐ

### カート獲得の確認
- セラーセントラル > 在庫管理 > 商品詳細 > 「カートボックス獲得の可能性」
- FBA出品でほぼ100%確保（自己発送は不利）

### 転売対策の3手順
1. **シリアルナンバー付与**: 商品に固有のシリアルナンバーを刻印/印刷
   - 転売業者が大量購入した際に追跡可能にする
2. **Amazonブランド登録**: 知的財産（商標）を登録 → 無断転売に対して申告権を取得
3. **継続監視**: 定期的に自社ASINを検索して相乗り出品・不正価格を検知

## STEP2: レビュー対策

**目標値（久野将司氏基準）:**
- スコア: 4.3以上（最低ライン4.0）
- 件数: 100件以上（初期目標50件）

### Amazon Vine（最優先）
- 出品直後に申請（レビュー0件の状態が最も効果的）
- 上限30件
- 手数料: 1〜2点無料 / 3〜10点¥10,000 / 11〜30点¥22,000
- ブランド登録が必要

### レビューリクエスト（自動化推奨）
- 購入5〜7日後に送る（早すぎると商品未到着、遅すぎると熱が冷める）
- Seller Sprite の一括レビューリクエスト機能を活用
- 禁止: レビューの内容を誘導する/インセンティブを提供する/否定的レビューを削除依頼

### レビュー獲得率の目安
- 自然発生: 0.5〜1%
- 施策実施時: 2〜4%

### 上位レビューの重要性（CVRへの影響）
- ユーザーは上位レビューから先に読む
- 「具体的な内容 × 写真/動画付き」の高評価レビューがCVR向上に直結
- セラーでは上位表示をコントロール不可 → 継続的に高品質レビューを獲得し続けることが重要

## STEP3: セール対策

**核心哲学: セールの「頻度」と「期間」を最大化する**

### なぜセールが重要か
- セール期間中はAmazonアルゴリズムの恩恵（インプレッション増）を受ける
- セール実施ごとにインプレッションの底が上がる（ただしCTR/CVR/レビューが整っている前提）
- 底上げが維持できないセラーの共通点: CTR/CVR/レビューが低い状態でセールを実施

### 3種のセール（優先順位順）
| 種類 | 費用 | 参加方法 | 期間 |
|------|------|---------|------|
| 特選タイムセール（DOTD） | 無料（今後有料化予定） | 招待型（Amazonから案内） | 大型セールイベントに準ずる |
| おすすめタイムセール（BD） | 手数料あり | セラーセントラルから作成 | 最大14日間 |
| 数量限定タイムセール | 手数料あり | セラーセントラルから申請 | 最大12時間（時間はAIが決定） |

### セールルール（必須厳守）
- 特選 ↔ おすすめ: **14日以上空ける**
- おすすめ ↔ おすすめ: **14日以上空ける**
- 数量限定 ↔ 数量限定: **7日以上空ける**
- **入稿順**: 先におすすめタイムセールを作成 → その後に特選をExcel入稿（2026年仕様変更）

### セールカレンダー（2025年開催実績）
重要度の高い順:
1. **プライムデー（7月）**: 年間最重要イベント
2. **プライム感謝祭（10月）/ ブラックフライデー（11月）**: 下半期の最大商戦
3. **初売りセール（1月）**: 年間SEOベースを底上げ（最重要な初動）
4. **新生活セール（2〜3月）**: 2026年より大型セール格上げ

### Camicks価格設計ルール
```
現在の通常価格: ¥2,480

【絶対禁止】クーポン使用
  理由: クーポン使用価格が「販売実績価格」として蓄積される
  → 次回セール時の基準が下がり、さらに深い割引が必要になる
  → 最終的にセール参加不可になるリスク
  → 立て直しには2〜3ヶ月必要

【推奨】セール価格探索（価格探索ゲーム）
  競合調査 → セール価格探索 → 通常価格探索の順
  目標: 「全体の利益を増やす値引き」として活用

【OK】販売実績価格に影響しない施策
  - ポイント率（デフォルト1%でOK）
  - ブランド割引（大型セール直後のカート放置層拾い上げに有効）
  - 定期おトク便（リピート率・LTV向上。プログラム参加推奨）
```

## STEP4: CTR対策

**核心哲学: メイン画像 = 「クリックを促すバナー画像」として設計する**

### CTRの確認方法
- Amazon Ads > ポートフォリオ > スポンサープロダクト広告 > CTR列
- **最低ライン: 0.3〜0.5%**（これを下回るとAmazonから徐々に露出削減）

### CTRが重要な2つの理由
1. **アルゴリズムの優遇**: CTRが高い = ユーザーの興味関心が高い → 広範なKWで表示されやすくなる
2. **広告費用対効果向上**: CTRが高い = CPCが下がりROASが向上。CTRが低い状態で広告調整しても効果なし

### メイン画像の絶対ルール（Amazonルール）
- 純白背景: RGB(255,255,255) — グレー・アイボリー完全NG
- 商品が画像の85%以上を占める
- テキスト・グラフィック・ロゴ・ウォーターマーク完全NG
- 商品に含まれない付属品・小道具は原則NG（ただしAI審査次第でグレーゾーンあり）

### CTR向上の4ポイント（久野将司氏のノウハウ）
1. **視覚的訴求力**: 中身が見える断面・立体感・ツヤ感（マット感よりツヤ感がCTR高い傾向）
2. **アテンションシール**: 訴求文言 × 女性顔写真で視線誘導
   - 縦長商品は横付けが効果的（シールと商品色を同化させると審査通過率UP）
3. **使用イメージ想起**: 小道具でも使用シーン訴求（AI審査で通れば有効）
4. **デジタル加工**: 立体感・ツヤ感を後処理で強調

### Camicksへの適用
- 現在のメイン画像: 白背景単体のシンプル設計（ルール遵守は完璧）
- 改善余地: アテンションシール追加を検討
  - 案例: 「外から見えない」「普通の靴下に見える」シール（横付け、商品色系）
- A/Bテスト活用（ほぼノーリスク）:
  攻めの新画像 → A/Bテスト設定 → 否認されてもA/Bが開始されないだけ（旧画像が表示継続）

### タイトル先頭30文字の設計
```
Camicks シークレット5本指靴下 ミドル丈 日本製 camifine® 和紙...
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
ここまでが検索結果で見える範囲
```
先頭30文字に必ず含めるもの: ブランド名（Camicks）/ 最大差別化KW（シークレット）/ カテゴリKW（5本指靴下）

## STEP5: CVR対策

**核心哲学: サブ画像1〜3枚目で大半のユーザーが離脱する → 権威性・信頼性を最初に持ってくる**

### CVR確認方法
- セラーセントラル > ビジネスレポート > **「ユニットセッション率」**（実質CVR）
- Amazon Ads: 注文数 ÷ クリック数（KW別に分析可能）
- 目標ROAS: 150〜200%以上（実務では主にROASを注視）

### CVR改善の優先順位（久野将司氏の定義）
1. **サブ画像**（最重要）: 1〜3枚目で大半が離脱。権威性・信頼性から始める
2. **上位レビュー**: 新しい・具体的・写真/動画付きが上位表示
3. **A+コンテンツ**: 読む人は5〜10%程度（Amazon担当者情報）。優先度は低め
4. **ストアページ**: ないよりマシ。とりあえず作ること

### サブ画像の訴求軸（Camicks推奨掲載順）
```
1枚目: 権威性・信頼性（ニッセンケン消臭試験数値 / 57年糸屋）← 離脱前に信頼を刷り込む
2枚目: 差別化（シークレット構造の視覚化 / 外から見えない証明）
3枚目: 機能（camifine和紙 / 調湿メカニズム / 酢酸90%削減）
4枚目: 理解促進（洗濯後裏返し不要 / 普通の靴下と同じ扱い）
5枚目: 優位性・比較（競合比較表 / 費用対効果）
6枚目: ブランドストーリー（57年製造背景 / 糸から一貫生産）
```

サブ画像の一般的な訴求軸カテゴリ:
- **権威性**: 資格保有者・専門家監修 / 実績（ランキング・受賞・メディア掲載）
- **信頼性**: 認定・認証 / エビデンス（試験結果）
- **理解促進**: 用途・使用例 / サイズ感 / 懸念の払拭
- **優位性/差別化**: 比較図表 / 費用対効果 / 製造背景

### ブランド分析の活用（ブランド登録後必須）
- **検索クエリパフォーマンス**: どのKWで流入・購入されているか（インプレッション→クリック→カート→購入のファネル）
- **マーケットバスケット分析**: あわせ買いデータ（クロスセル判断・新商品開発ヒント）
  - 自社商品同士の組み合わせ率12%超 → セット販売を検討

## STEP6: SEO対策

**核心哲学: テキストSEOは「前提条件」。販売実績こそが検索順位を決める**

### Amazon SEO 3大要素
1. **販売実績**（最重要）: 検索KWごとの販売数。外部流入経由の販売実績はSEO高評価
2. **関連性**: タイトル・説明文にKWを自然に含める（インデックスの前提）
3. **顧客体験**: 在庫切れゼロ・FBA利用・レビュー評価

### 在庫管理（SEOの土台）
- **常に1.5〜2ヶ月分の在庫を確保**
- 大型セール（プライムデー/ブラックフライデー）では平常時より販売速度が大幅UP → 2〜3倍確保
- **在庫切れのリスク**: 即圏外 + 補充後も元の順位に戻りにくい
- **FBA一択**（大型商材除く）: Amazon優遇 + 配送正確性 + 顧客体験向上

### KWインデックス確認方法
- Amazonの検索窓に「ASIN + KW」で検索 → ヒットすればインデックス済み
- インデックスされていない場合:
  → タイトル・説明文にKWを自然に含める
  → SP広告で該当KWを配信（インデックス目的として有効）

### 注力KW選定（Camicksの場合）
目安10個。選定基準:
- ✅ 一定の検索ボリュームがある
- ✅ 実際に購入されている（ブランド分析 or Ads レポートで確認）
- ❌ 商品との関連性が低い
- ❌ 商品特性と異なる（誤期待を生む）

### KW調査ツール
| ツール | 特徴 | 用途 |
|--------|------|------|
| ブランド分析 | 一次情報・無料 | 検索クエリパフォーマンス / 購入KWの確認 |
| Amazon Ads レポート | 無料 | 広告経由の購入KWとROAS |
| Seller Sprite | 有料（割引コード: QOI116で30%OFF） | 競合KW逆引き / 検索ボリューム / キーワードモニタリング |
| Keepa | 無料（拡張）/ 有料 | 価格推移 / セール頻度 / BSRトレンド |

### SEO加速施策（外部流入が最強）

**外部流入経由の購入はSEO高評価**（Amazonが外部からの集客を優遇）

施策①: **ファンくる**（応募型モニター購入プラットフォーム）
- 主目的: 注力KWの販売実績を短期間で積む
- 副目的: レビュー誘導（付与率5〜7%）/ リピーター獲得
- 費用シミュレーション（Camicks ¥2,480 × 150件 × 80%還元）:
  ```
  初期費用（固定）: ¥150,000
  アサイン費用（800円 × 150件）: ¥120,000
  商品代金還元（2,480円 × 150件 × 80%）: ¥297,600
  合計支出: ¥567,600（+税）
  Amazon売上（2,480円 × 150件）: ¥372,000
  実費（売上控除後）: ≈ ¥195,600
  ```
  → 約20万円で150件の販売実績を短期積上げ
- 謝礼: 80%還元推奨（100%は質の低いユーザーが集まる / 70%以下は応募数が少ない）
- 注意: 閑散期・競合が弱い時期に使うのが最効率

施策②: **SNS広告**（Meta広告 / X広告 / LINE広告 → Amazon送客）
- 送客のポイント: セール訴求「セール中」/ 限定訴求「Amazon限定」
- デメリット: 自社EC流入が減る可能性。全体最適で判断する

施策③: **リストマーケティング**（公式LINE / メルマガ → Amazon）

### KW検索順位モニタリング
- 頻度: 週次
- ツール: Seller Sprite のキーワードモニタリング / 手動（Chromeシークレットモードで確認）
- 確認内容: 検索順位 / CVR / ブランドシェア（購入ブランドシェア > インプレッションシェアなら「選ばれている」証拠）

## STEP7: 広告対策

**核心哲学: 「レビューが整う前に広告費を投じても採算が取れない」**
最初は小さく配信してデータを蓄積 → 徐々に拡張

### 広告の2つの目的
1. **露出面を広げる（攻め）**: スマホFVは広告枠しか見えない / 競合商品ページにも表示可能
2. **顧客の離脱を防ぐ（守り）**: 指名KW・自社商品ページに自社広告を置いて競合参入を防ぐ

### 広告3種類と優先順位
| 種類 | 目的 | 割合（目安） |
|------|------|------------|
| **SP（スポンサープロダクト）** | 売上最大化 | 8〜9割 |
| SB（スポンサーブランド） | ブランド・商品認知 | 補助 |
| SD（スポンサーディスプレイ） | リターゲティング | 補助 |

→ **SPに集中することが効率的**。SB/SDはROASが合いにくい

### 6キャンペーン構成（久野将司氏の実務フォーマット）

```
001_01_SP_Camicks_オート
  目的: KW宝探し・幅広い配信
  設定: 入札50円以下 / ターゲティンググループ4種全ON
  運用: お宝KWを発掘 → マニュアル（03〜06）に移行

001_02_SP_Camicks_指名KW
  目的: ブランド指名検索での顧客離脱防止（守りの運用）
  設定: 完全一致+フレーズ一致のみ / メンテナンス頻度低め
  運用: 注力ロングテールKWは別分けして入札UP

001_03_SP_Camicks_一般KW
  目的: 比較検討層へのアプローチ（最も売上ポテンシャル大）
  設定: 関連KW10個を中心に配信
  注意: ビッグワードはインプレッションだけ伸びて予算が溶けやすい

001_04_SP_Camicks_競合商品
  目的: 競合商品ページに広告を配信し競合顧客を獲得（攻め）
  設定: 入札50円以下 / 300〜500 ASIN登録（Helium 10で一括コピー推奨）
  注意: 商品ターゲットにKWが混入するバグに注意 → 除外設定

001_05_SP_Camicks_クロスセル
  目的: 自社ライン商品間での相互流入 + 競合商品への離脱防止
  設定: 同ブランドラインの全商品を登録

001_06_SP_Camicks_カテゴリー
  目的: お宝カテゴリーの発掘（拡張フェーズ）
  設定: 入札50円以下 / 01〜05が順調になってから着手
  メリット: 上手く運用すれば低CPC高ROAS（例: 400%）
```

### 入札設計（重要）
| 設定 | 推奨値 | 理由 |
|------|--------|------|
| 入札戦略 | **ダウンのみ** | アップとダウンはCPC高騰リスク / 固定額は平均CPC高くなる傾向 |
| 掲載枠調整 | **追加課金0%** | レビューが整えば追加課金なしで自然に上部表示 |
| 初期入札額 | **100円以下から開始** | 2,000〜3,000円配信してROASを確認してから調整 |

### 入札チューニング基準
