## 1. Amazonランキングを決める7要素（重要度順）

| 優先度 | 要素 | 説明 | Camicks目標値 |
|--------|------|------|--------------|
| ★★★★★ | **転換率 (CVR)** | 商品ページを見た人が買う率。最重要KPI | 12〜15% |
| ★★★★★ | **販売速度** | 単位時間の売上数。初動が命。出品直後の速度がランキングを決める | 初月100足/週 |
| ★★★★☆ | **CTR（クリック率）** | 検索結果サムネイルから商品ページへのCTR | 5%+ |
| ★★★★☆ | **KW適合度** | タイトル × 説明 × バックエンドKWのKW密度 | タイトルに主要KW3語+ |
| ★★★☆☆ | **レビュー評価** | 件数 × 星数。4.0以上で検索優遇。4.5以上で信頼構築完成 | 4.5以上 / 50件以上 |
| ★★★☆☆ | **在庫** | 在庫切れ = 即圏外。FBA推奨。安全在庫300足以上 | FBA + 300足在庫 |
| ★★☆☆☆ | **価格競争力** | Buy Box獲得率。FBA出品でほぼ100%確保できる | ¥2,480維持 |

**核心原則**: AmazonはAmazonが儲かる商品を上位表示する。
= 「CVRが高い × 販売速度が速い = Amazonが儲かる」 = ランキング上昇。

## 2. Camicksの転換率設計（月商1,000万円からの逆算）

```
月商目標: ¥10,000,000
÷ 販売単価: ¥2,480
= 必要販売数: 4,032足/月

÷ 転換率目標: 15%
= 必要セッション数: 26,880/月

÷ 30日
= 必要日次セッション: 896/日（≈ 900/日）

つまり: 日次900セッション × CVR15% = 134足/日 → 月4,032足達成
```

**CVR15%達成の3要素:**
1. **メイン画像**（検索CTRを決める）: 白背景 + 商品85%以上 + 高コントラスト
2. **サブ画像アーク**（ページ内CVRを決める）: 機能→根拠→価格正当化→感情の流れ
3. **価格 × レビュー数**（信頼を決める）: ¥2,480は高額帯 → 根拠画像必須

## 3. 検索CTRを決める要素（サムネイル設計）

検索結果でユーザーが見るのは **2つだけ**: メイン画像 + タイトル先頭30文字。

### メイン画像の絶対ルール（Amazonルール）
- 純白背景: RGB(255,255,255) — グレー・アイボリー・クリーム完全NG
- 商品が画像の85%以上を占める
- テキスト・グラフィック・ウォーターマーク完全NG
- 商品が完全に映っている（切れNG）

### サムネイル視認性（120px換算）
- 深いネイビーの靴下 × 白背景 = コントラスト最大
- 靴下は「立体感」が伝わる角度で撮影
- 2足セットで「ペア感」「premium感」を出す

### タイトル先頭30文字の設計
```
Camicks シークレット5本指靴下 ミドル丈 日本製 camifine® 和紙...
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
ここまでが検索結果で見える範囲
```
先頭30文字に必ず含めるもの:
- ブランド名（Camicks）
- 最大差別化KW（シークレット）
- カテゴリKW（5本指靴下）

## 4. タイトル最適化の公式

```
[ブランド名] + [最大差別化KW] + [カテゴリKW] + [主要機能] + [サイズ/対応] + [素材特徴]
```

**Camicksのタイトル公式（具体例）:**
```
Camicks シークレット 5本指靴下 ミドル丈 日本製 camifine® 和紙 消臭 蒸れない
メンズ レディース 兼用 23-27cm 縫い目ゼロ 外から見えない 五本指ソックス
```

タイトル最適化のルール:
- 最大文字数: 全角80文字（Amazonの表示限界）
- 記号（/・｜）は使ってよい。スペースで区切る
- 大文字ブランド名は先頭のみ
- 効能・薬機法NG表現は入れない（「水虫」「外反母趾に効く」等）

## 5. バックエンドKW設計の原則

| 項目 | ルール |
|------|--------|
| 文字数 | 250バイト（全角125文字）以内 |
| 重複 | タイトル・箇条書きに既にある語は入れない（重複カウントなし） |
| スペース | 区切りはスペースのみ。カンマ・読点NG |
| 口語体 | 検索されやすい自然な言葉で（「靴下おすすめ」「足指 靴下 普通に見える」等） |
| KW選定基準 | 検索ボリューム > 競合密度（ニッチKWで上位取る戦略） |

**Camicks優先バックエンドKW候補（タイトル未使用語）:**
```
足指靴下 普通に見える 靴を脱いでも バレない 5本指 足臭 蒸れ対策
和紙素材 camifine 酢酸 イソ吉草酸 試験データ 縫い目 ホールガーメント
57年 糸屋 職人 靴下 洗濯 裏返し不要 おしゃれ 室内 スタイリッシュ
```

## 6. Amazon攻略7ステップ（ロードマップ）

### Step 1: 商品登録の最適化
- タイトル（上記公式通り）
- 価格設定（競合比較 × 価格正当化 × 初動割引設計）
- FBA登録（Buy Box確保 + 配送品質でレビュー安定化）
- カテゴリ選定（「ソックス」より「5本指靴下」がニッチで上位取りやすい）

### Step 2: コンテンツ最適化
- **メイン画像**: 白背景 × 商品85% × 高コントラスト（CTR直結）
- **サブ画像6枚**: 機能→根拠→価格正当化→感情の完全なアーク
- **商品説明文**: 3層KW × 薬機法クリア × 価格正当化ストーリー
- **A+コンテンツ**: ブランドストーリー × 素材科学 × 競合比較表

### Step 3: KW最適化
- バックエンドKW250文字（重複なし・口語体）
- タイトルKW配置（先頭30文字に最重要KW）
- 箇条書きにもKW自然配置（詰め込みすぎNG）

### Step 4: レビュー獲得戦略
- **Vine申請**: 出品直後に申請。30件まで無料でレビュー獲得可能
- **フォローアップメール**: 購入7日後に送る（Amazonの許可範囲内）
- **Q&A初期対応**: 質問が来たら24時間以内に回答 → 信頼構築
- **目標**: 初月30件（Vine活用）→ 3ヶ月で100件

### Step 5: 広告戦略（スポンサードAD）
- **フェーズ1（出品〜1ヶ月）**: オートキャンペーン × 予算¥1,000/日
  - 全部Amazonに任せて「何で検索されているか」データを収集
- **フェーズ2（1ヶ月後〜）**: マニュアルキャンペーン × 勝ちKWに集中
  - オートキャンペーンの「購入につながったKW」をマニュアルに移行
- **ACOS目標**: 30%以内（売上¥2,480 × 30% = ¥744まで広告費OK）
- **ROAS目標**: 300%以上

### Step 6: 価格戦略
- **カート獲得**: FBA出品でほぼ100%確保。価格は¥2,480固定
- **タイムセール**: 月1回実施。15〜20%OFF × 期間3日
  - 目的: 初動の販売速度UP → ランキング押し上げ → オーガニック流入増
- **クーポン設定**: 10%OFFクーポン常時設置（CTRが約15%UP）

### Step 7: PDCA（週次継続）
週次で以下KPIを確認 → 最も乖離しているKPIに「must_fix #1」指定:
- Sessions（セッション数）目標: 900/日
- Unit Session % （転換率）目標: 12〜15%
- Buy Box % 目標: 95%+
- Organic Rank: 「シークレット 5本指靴下」で TOP20以内
- Review velocity: 週5件以上

## 9. サブ画像アーク設計の公式（CVR最大化）

```
メイン（白背景）
  ↓
サブ①: シークレット訴求（Layer1: シークレット追求層が心を掴まれる）
  ↓
サブ②: 機能説明（Layer2: 機能健康層が「なるほど」となる）
  ↓
サブ③: 証拠・権威（Layer2/3: ニッセンケン数値で信頼構築）
  ↓
サブ④: 利便性訴求（Layer1/2: 洗濯後裏返し不要 → 日常コスト低い）
  ↓
サブ⑤: 製造ストーリー（Layer3: ブランド信頼層が「本物だ」となる）
  ↓
サブ⑥: 感情・着用シーン（全層: 「自分もこうなりたい」のイメージ訴求）

このアークで:
- 機能説明枚（サブ①②③）: 「買う理由」を作る
- 根拠・権威枚（サブ③⑤）: 「信頼できる」を作る
- 感情訴求枚（サブ①④⑥）: 「欲しい」に変換する
```
