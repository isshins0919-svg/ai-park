# さくら台本設計ガイド

## キャラクター前提

- **さくら** = プロのウグイス嬢（野球場）
- **裏アカ** = 深夜のベッドで一人、発声練習している「素の姿」を見せる
- **顔は映さない** — 首から下 or 後ろ姿で統一

---

## 台本の3型

### 型1: 発声練習型（基本）

「今日の練習をしている」という体裁。プロの声 × 深夜のベッド × 部屋着のギャップ。

**構造:**
1. プロ口調でアナウンスを始める（例: 「バッターボックスに入りましたのは——」）
2. 途中で素の独り言が漏れる（例: 「...あ、ここいつも噛むんだよな」）
3. またプロ口調に戻る
4. 最後に素の声で一言（例: 「...もう寝よ」）

**バリエーション例:**
- スタメン発表の練習
- ホームラン実況の練習（小声ver = ASMR寄り）
- 場内アナウンスの練習（「本日の試合は〜」）
- ヒーローインタビューの練習

### 型2: 本音漏れ型

「球場では言えないこと」を深夜に一人でつぶやく。感情的でファン化を狙う。

**構造:**
1. 仕事の一場面を回想する口調で始める
2. そこで感じた本音を吐露する
3. 感情が高まる瞬間を入れる
4. 最後にプロ口調に切り替えて締める（ギャップ）

**バリエーション例:**
- 4番バッターに言えなかったこと
- 声が裏返った日の恥ずかしさ
- 引退試合で泣きそうになった話
- 選手の名前を間違えそうになった瞬間

### 型3: ASMR練習型

「明日の試合に向けて囁き練習」という設定。純ASMR体験。

**構造:**
1. 「明日の準備しよ...」と囁きで入る
2. ウグイス嬢のセリフを極限まで小さな声で繰り返す
3. 時々素の感想が漏れる（小声のまま）
4. 最後に急にプロ声量で一言 → 「...あ、うるさかったかな」で落とす

---

## 台本共通構造（60秒以内）

```
[0-3秒]  フック — スクロール停止させる冒頭
[3-45秒] 本体 — 型に沿ったコンテンツ
[45-50秒] 間 — 2-3秒の無音（壺ポイント）
[50-60秒] 締め — ウグイス嬢口調に戻す or 素の一言
```

---

## 壺ポイント（必ず入れる）

1. **2-3秒の無音** — セリフの後に意図的に沈黙を入れる。この「間」が一番ドキドキさせる
2. **最後の一言でギャップ** — 急にウグイス嬢口調に戻す。この切り替えで保存・フォローが起きる
3. **息づかい** — セリフの間に自然な息づかいを入れる。生々しさの演出
4. **独り言の漏れ** — 台本の途中で「...あ、」「...ん、」など素の反応を挟む

---

## NG表現（絶対に入れない）

- 直接的なエロ表現 → TikTokのBANリスク + ミステリアス感が壊れる
- 「フォローしてね」等の直接CTA → 世界観が崩れる
- 特定の球団名・選手名 → 権利問題回避
- 顔が特定できる情報 → 匿名性の維持

## フック設計パターン

| パターン | 例 |
|---|---|
| 状況提示型 | 「深夜2時。明日の試合の練習してる。」 |
| 疑問喚起型 | テロップ「ウグイス嬢が深夜にベッドで何してると思う？」 |
| 音から入る型 | いきなりプロのアナウンス声 → 画は部屋着のベッド |
| ギャップ即出し型 | プロ声で1秒 → 急に素の小声「...あ、噛んだ」 |

## 品質ゲート（5段階）

台本完成後、以下の5軸で100点満点評価を行う:

1. **フック強度** (20点) — 最初の3秒でスクロールが止まるか
2. **テンポ** (20点) — 3秒ごとに新情報があるか、中だるみがないか
3. **離脱防止** (20点) — 途中で「もういいや」と思わせるポイントがないか
4. **ギャップ演出** (20点) — プロ声⇔素の声の切り替えが効果的か
5. **ループ性** (20点) — 最後→冒頭がつながるか、もう一度見たくなるか

**合格ライン: 75点以上**
