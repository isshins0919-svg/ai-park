## Meta（Instagram / Facebook）Reels 安全帯

### 危険ゾーン（テロップ禁止エリア）
| エリア | ピクセル範囲 | y_ratio換算 | 理由 |
|--------|------------|------------|------|
| 上部 | 0〜108px | y_ratio < 0.056 | アカウント名・フォローボタン |
| 下部 | 1600〜1920px | y_ratio > 0.833 | いいね・コメント・シェアUI + URLバナー |

### 安全帯（テロップ推奨エリア）
| 用途 | y_ratio | 備考 |
|------|---------|------|
| HOOK・強調 | 0.55〜0.62 | 顔の下、UI干渉ゼロ |
| 通常説明 | 0.63〜0.70 | 中央よりやや下 |
| OFFER・CTA | 0.68〜0.75 | UIより上、CTAクリック導線上 |
| 固定注釈（薬機法） | y_start=24px (上部固定) | 上部アカウント表示と重なるが小文字で許容 |

### edit_ai_v2.py の設定値
```python
# make_telop() 内 dynamic_y 推奨値
fs >= 80  → dynamic_y = 0.62  # 大きい（HOOK・強調）
fs >= 65  → dynamic_y = 0.67  # 中くらい（通常）
else      → dynamic_y = 0.72  # 小さい（OFFER・CTA）

# 安全帯フロア（最低限この値以上）
effective_y = max(raw_y, 0.55)  # 0.75は誤り（Meta下部UI干渉ゾーンに入る）
```

## DPro分析 — 美容・スキンケアジャンルの勝ちパターン（2026-04取得）

### Instagram上位広告の傾向（genre_id=80相当）
| 分析項目 | 傾向 |
|--------|------|
| テロップY位置 | 画面中央〜下60〜75%帯に集中 |
| テロップスタイル | 白文字+黒縁が基本、強調時は黄色orオレンジ |
| フォントサイズ | 本文60〜80px、強調90〜110px |
| テロップ密度 | 1行完結が基本、最大2行まで |
| 参考ブランド | オルビス、アテニア、キミエリンクルホワイト |

### 推奨テロップスタイル体系（6スタイル）
| スタイル名 | 用途 | 文字色 | 背景 | 強調 |
|-----------|------|--------|------|------|
| `impact` | HOOK・冒頭フック | 白 | 黒半透明バー | ✅ |
| `default` | 通常説明 | 白 | なし（縁のみ） | - |
| `emphasis` | 数字・キーワード強調 | 黄色 | なし | ✅ |
| `lower_third` | 商品名・固有名詞 | 白 | 半透明帯 | - |
| `proof` | 実績・信頼数字 | 白 | ネイビー背景 | ✅ |
| `cta` | CTA・行動喚起 | 白 | 赤/オレンジ背景 | ✅ |

## 注意事項

- **Meta Reelsの下部320px（y_ratio > 0.833）は絶対禁止** — URLバナー・UIボタンと重複
- **テロップの「下寄せ」は参考動画の白レターボックス方式を模倣したもの** — Metaでは逆効果
- **次回動画提案時は必ずDPro上位3本の分析を含める**（同ジャンル・同プラットフォーム）
- **安全帯チェックリスト**: 上108px✅ / 下320px✅ / テロップ1行完結✅ / 縁あり✅
