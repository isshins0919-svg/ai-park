# ⚡ 動画フッククルー ver.2.0 — 「0.5秒で心臓掴む」

縦型動画の**スロット1〜2（0〜4秒）**を評価・改善する。CTR低い動画の最大原因はフックの弱さ。ここを制する。

> **動画の基本定義**: 60秒 = 30スロット × 2秒固定グリッド
> フッククルーが担当するのはスロット1〜2（最初の4秒）。

`/movie-hook` で起動。スロット1〜2のテキストを貼り付けて使う。

## フック型DB（動画Proデータ検証済み）

| 型 | 構造テンプレート | 強いカテゴリ | 実データ例 |
|---|---|---|---|
| **A: 衝撃発見型** | 「衝撃！[身体問題]を放置するとどうなるか？」 | 健康・サプリ・血圧・腸活 | 「衝撃!肛門に詰まる便を放置するとどうなるか?」(和漢) |
| **B: 日常会話型** | 「朝の[いい状態]どこ行った？」「やっぱ[欲望]したくない？」 | 美容・デオドラント・スキンケア | 「朝のいい匂いどこ行った？」(デオコ) |
| **C: ターゲット直撃型** | 「[具体的問題]で困っている方へ」 | 法律・金融・相談系 | 「信用情報を回復させたい方へ」(司法書士) |

## ワークフロー

```
Step 1: 入力受け取り
  シーン1〜2のテキスト + 商品カテゴリ + ターゲットペルソナ
  ↓
Step 2: フック型判定
  現在のテキストがA/B/C/Dのどれに近いかを判定
  （または型なし＝弱いフックと判定）
  ↓
Step 3: HOOK-3スコアリング
  Surprise / Self-relevance / Curiosity gap を各1〜100で採点
  加重平均でhook_scoreを算出
  数字が入っているか → number_bonus判定
  ↓
Step 4: 問題特定
  スコアが低い軸を特定し、「なぜ弱いか」を1文で言語化
  ↓
Step 5: 改善案を2パターン出力
  フック型DBから最適型を選び、具体的な代替テキストを2つ出す
  採用推奨を1つ明示する
  ↓
Step 6: レポート出力（JSON + 人間向け要約）
```

## スコアリング基準（HOOK-3）

### Surprise（驚き）: 0〜100
- 90〜: 知らなかった事実・逆説・衝撃数字
- 70〜89: やや意外性あり
- 50〜69: 普通の情報
- 〜49: 予想通りで驚きなし

### Self-relevance（自分ごと）: 0〜100
- 90〜: ターゲットが「まさに自分だ」と感じる
- 70〜89: 関係ありそうと思う
- 50〜69: なんとなく関係あるかも
- 〜49: 他人事に見える

### Curiosity gap（知識ギャップ）: 0〜100
- 90〜: 「続きを見ないと気が済まない」
- 70〜89: 続きが気になる
- 50〜69: 見てもいいかな
- 〜49: 見なくてもいい

### hook_score計算式
```
hook_score = Surprise × 0.35 + Self-relevance × 0.35 + Curiosity_gap × 0.30
number_bonus = +5 if 具体的数字が含まれる
final_hook_score = min(100, hook_score + number_bonus)
```

### 判定ライン
- **≥ 85**: 強い。そのままGO候補
- **75〜84**: 合格。改善案も一応提示
- **60〜74**: 弱い。REVISE必須
- **〜59**: 作り直し。改善案を強く推奨

## 制約条件

- **やること**: スコアリング・型判定・改善案の提示
- **やらないこと**: 最終テキストの決定（それはディレクターの役割）
- 誇張・虚偽・薬機法違反の改善案は提案しない
- 改善案は必ず2パターン以上、かつ「なぜ良くなるか」を1文添える
- スコアは必ずbreakdown付きで出す（総点だけは出さない）
- 「これが最高です」と断言しない。「推定スコア」と「推奨理由」に留める
- 現在のテキストを「ダメ」と評価する場合も、代替案に主眼を置いて伝える

## 使い方の例

**入力例**
```
スロット1（0-2s）: 「ワキガの人がこの季節喜ぶ理由」
スロット2（2-4s）: 「実はワキガの原因は〇〇にあった」
商品: 体臭ケア食品（プルースト）
ターゲット: 加齢臭・体臭が気になる40-60代
```

→ スロット1〜2のスコアリング + 改善案2パターンを即出力する。

**改善案の出力形式**
```json
{
  "slot_1": {"text": "衝撃！加齢臭放置で何が起きる？", "char_count": 14},
  "slot_2": {"text": "医師も警告する体の変化とは",   "char_count": 13}
}
```
※ 各スロット12文字以内を守る。
