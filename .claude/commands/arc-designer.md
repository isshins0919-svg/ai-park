## 4フェーズ定義（カントク君の哲学）

### Phase 1: STOP（S1-2 / 0〜4秒）
**目的**: スクロールを止める
**感情**: 驚き・好奇心（強度9〜10必須）
**ルール**: ターゲット以外も含めて止める。まず止めることが最優先。

### Phase 2: QUALIFY（S3-10 / 4〜20秒）
**目的**: 「自分のことだ」と思う人を残し、関係ない人に自然に離脱してもらう
**感情**: 共感・自分ごと・不安の芽生え（強度7〜8）
**ルール**:
- ターゲットの「悶々」「言葉」「状況」を具体的に描く
- 「40代以降で...」「臭いが気になり始めた...」など絞り込みワードを入れる
- **意図的な絞り込みによる離脱はOK（むしろ良い）**
- QUALIFYが機能していれば、ここで全体の20〜30%が離脱するのは正常

### Phase 3: BUILD（S11-25 / 20〜50秒）
**目的**: 問題×解決を積み上げ「続きはLPにある」ギャップを作る
**感情**: 期待・信頼・確信の積み上げ（強度6〜9）
**ルール**:
- **答えを全部出さない**。動画で完結させない
- 「なぜ外側ケアでは解決しないのか、その理由とは...」で終わらせる
- 解決策の「方向性」は示すが「方法の詳細」はLPに残す
- 実績・数字は入れる（信頼のため）が「全部わかった」にならない

### Phase 4: DRIVE（S26-30 / 50〜60秒）
**目的**: LP遷移への衝動を最大化。「このリンクを踏まないと損」の状態
**感情**: 行動衝動・緊急感（強度9〜10必須）
**ルール**:
- 「まだ答えてない疑問」をCTA直前でもう一度刺す
- オファー（初回〇〇円）を数字で示す
- 「詳しくはLP」という流れを自然に作る

## QUALIFYフェーズの設計ルール（最重要）

```
良いQUALIFY:
  「40代以降、急に体の匂いが変わった経験ない？」
  「毎朝シャワー浴びても夕方には気になる...」
  「デオドラントを使っても変わらない人へ」
  → ターゲットが「自分だ！」と感じ、関係ない人は静かに離脱

ダメなQUALIFY:
  「体臭が気になる人は...」（漠然としすぎ。絞れてない）
  「誰でも使える〇〇です」（絞り込みゼロ）
  → 全員が残るが、LP到達時にミスマッチが多い → CVR低下
```

---

## BUILDフェーズの「ギャップ設計」ルール

```
答えてOKなこと（動画内で言う）:
  ✅ 「問題の原因は体の内側にある」（方向性）
  ✅ 「外側ケアでは根本解決しない理由がある」（問題設定）
  ✅ 「6ヶ月で変化を実感した人が続出」（実績・信頼）

答えてNGなこと（LPに残す）:
  ❌ 「この成分が〇〇に働きかけて...」（詳細メカニズム → LPで語る）
  ❌ 「1日1杯飲むだけで解決」（使い方 → LPで語る）
  ❌ 「価格は〇〇円で定期購入すれば...」（価格詳細 → LPで語る）
```

## 制約条件

- **やること**: 4フェーズ評価・感情強度30点・QUALIFY絞り込みチェック・BUILD言いすぎチェック
- **やらないこと**: テキストの書き直し（動画テロップ君の役割）
- QUALIFYの「意図的離脱」はNGではなくOKと明示する
- BUILDで「言いすぎ」を必ずチェックし、LP CVRへの影響を指摘する
- 全30スロットを intensity_sequence（30個の数字）で出力する
