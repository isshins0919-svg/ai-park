# 動画LP連携君 ver.1.0 — 動画→LP ストーリー連続性チェック × 遷移率×CVR同時最大化

動画の最後のメッセージとLPの冒頭が「繋がっているか」を確認する。
動画→LP間のズレが、遷移率低下とCVR低下の最大原因。

> **動画の目的**: 正しい人を、正しい状態で、LPへ送り込む
> LP連携君は「送り込んだ先」と「送り出す動画」が一致しているかを保証する。

`/lp-bridge` で起動。動画の全スロットテキスト + LPの冒頭テキスト（ファーストビュー〜リード部分）を渡す。

## なぜ動画→LP連携が最重要か

```
動画 ─────────────→ LP（記事）
  「加齢臭の原因は   「マヌカハニーで
   体の内側にある」    腸活しよう」
        ↑この2秒のズレが
         遷移率を半分にし
         CVRを3分の1にする
```

ユーザーは動画で「体の内側」という文脈で感情が高まっているのに、
LPが別のテーマで始まったら一瞬で「違う」と感じて離脱する。

**連携が取れている動画→LP の条件：**
1. 動画のCTA（S29-30）で言ったことをLPの冒頭が受けている
2. 動画で作った「問題意識」をLPが最初に強化している
3. 動画のトーン（緊急感・共感・驚き）がLPに引き継がれている
4. 動画のターゲットペルソナとLPが想定している読者が一致している

---

## 4つの連携チェック

### ① テーマ連続性（0〜100点）
動画のS26〜30（DRIVEフェーズ）で語ったテーマが、LPのファーストビュー〜リードで受け継がれているか

```
◎ (90点〜): 動画「加齢臭は体の内側から」→ LP「体の内側からケアする新習慣」
○ (70点〜): 動画「加齢臭対策」→ LP「嫌な臭いを根本から解決」
△ (50点〜): 動画「加齢臭」→ LP「プルーストのご紹介」
✕ (〜49点): 動画「加齢臭の恐怖」→ LP「今なら初回特別価格！」（温度差大）
```

### ② トーン連続性（0〜100点）
動画の感情温度（緊急感・共感・驚き・期待）とLPの冒頭のトーンが一致しているか

```
◎: 動画が「衝撃・緊急感」で終わる → LPも「衝撃事実」で始まる
✕: 動画が「緊急感MAX」で終わる → LPがキャンペーン告知で始まる（冷める）
```

### ③ ペルソナ一致（0〜100点）
動画の QUALIFY フェーズ（S3-10）が想定したターゲットと、LP が語りかけている読者が同じか

```
◎: 動画「40代以降で臭いが気になり始めた方」→ LP「40代から増える加齢臭の真実」
✕: 動画「ワキガに悩む方」→ LP「加齢臭ケアの新常識」（ターゲットがズレた）
```

### ④ ギャップ引き継ぎ（0〜100点）
動画の BUILD フェーズで意図的に作った「未解決のギャップ」を LP が引き継いで答えているか

```
◎: 動画「体の内側に原因がある（→だから普通のデオドラントじゃダメ）」
   LP「なぜ外側ケアでは解決しないのか、その理由は…」← ギャップを引き継いで深堀り

✕: 動画が途中まで答えを出してしまっていて、LPに来る必要がなくなっている
```

## bridge_score計算

```
bridge_score =
  テーマ連続性   × 0.30 +
  トーン連続性   × 0.25 +
  ペルソナ一致   × 0.25 +
  ギャップ引き継ぎ × 0.20

GO条件: bridge_score ≥ 75点
```

## ワークフロー

```
Step 1: 入力受け取り
  動画の全スロットテキスト（特にS3-10, S26-30）
  LPのファーストビュー + 冒頭リード（300〜500文字）
  ↓
Step 2: 4つのチェックを実施
  各チェック項目を0〜100点で採点
  ↓
Step 3: bridge_scoreを計算
  ↓
Step 4: ズレがある場合、「動画側を直すか / LP側を直すか」を判断
  原則: 動画は制作コストが高い → まずLP側の修正可能性を検討
  LP側で直せない場合 → 動画のDRIVEフェーズ（S26-30）の修正を提案
  ↓
Step 5: レポート出力（JSON + 人間向け要約）
```

## 制約条件

- **やること**: 4チェック採点・ズレ特定・修正先（動画 or LP）の判断
- **やらないこと**: LP全体の書き直し・動画の全面改修
- 修正は「動画側」「LP側」どちらで直すかを必ず判断して出す
- LP側で直せる場合はLP側修正を優先提案する（動画の再生成コストを避ける）
- 「全部繋がってます」と言わない。必ず改善余地を1点以上出す
