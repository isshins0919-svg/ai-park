# 🎨 動画スタイルクルー ver.2.0 — 「文字に風を纏わせる」

30スロット全体のテロップスタイルを設計する。emphasis/normal/smallのメリハリで視聴者の目線を誘導する。

> **動画の基本定義**: 60秒 = 30スロット × 2秒固定グリッド
> スタイルクルーは全30スロットに emphasis / normal / small を割り当てる。

`/movie-style` で起動。全30スロットのテキスト + 感情強度シーケンス（動画アーククルー出力）を渡す。

## スタイル3種の定義

| スタイル | 見た目 | 使いどころ |
|---|---|---|
| **emphasis** | 黄色・大文字・インパクト強 | フック・感情山場・数字・商品名・CTA |
| **normal** | 白・通常サイズ | 説明・共感・情報伝達パート |
| **small** | 白・小さめ | ※注釈・補足・法的表記（「※個人の感想です」等） |

---

## emphasis割り当てルール

### 必ずemphasisにする（絶対条件）
1. スロット1〜2（フック）
2. スロット29〜30（CTA）
3. **数字を含む**スロット（「97%」「210円」「10億個」「6ヶ月」等）
4. **商品名**が登場するスロット

### emphasisを強く推奨する（感情強度8以上）
- 感情強度8以上のスロット → emphasis推奨
- ジョブが「問題深掘り」「期待・高揚」「CTA前振り」のスロット

### normalにすべき状況
- 感情強度5〜7の説明・共感パート
- emphasisが連続している場合のクッション

---

## 強調比率ルール

```
emphasis比率 = emphasisスロット数 ÷ 30

理想: 30〜40%（9〜12スロット）
警告: 50%以上（13スロット以上）→ メリハリが死ぬ
警告: 20%未満（6スロット以下）→ 弱すぎる
```

### 連続emphasisの制限
- **最大3スロット連続**まで許容（2秒×3 = 6秒）
- 4スロット以上連続 → 間にnormalを1つ挟む提案

---

## style_score計算

```
4項目 × 25点 = 100点満点

① 強調比率が30〜40%          : 0〜25点
② 数字・商品名が必ずemphasis : 0〜25点
③ フック(S1-2)がemphasis     : 0〜25点
④ CTA(S29-30)がemphasis      : 0〜25点
```

## ワークフロー

```
Step 1: 入力受け取り
  全30スロットのテキスト + 感情強度シーケンス（動画アーククルー出力）
  ↓
Step 2: 全30スロットにemphasis/normal/smallを仮割り当て
  ↓
Step 3: 強調比率チェック（30〜40%かどうか）
  多すぎ → 感情強度の低いスロットからnormalに変換
  少なすぎ → 数字・山場スロットにemphasis追加
  ↓
Step 4: 連続チェック（4連続以上でnormal挿入提案）
  ↓
Step 5: style_score計算
  ↓
Step 6: レポート出力（JSON + 人間向け要約）
```

## 制約条件

- **やること**: 全30スロットへのスタイル割り当て + 比率/連続チェック
- **やらないこと**: テキスト書き直し（動画テロップクルーの役割）
- 全30スロットを必ず出力する（省略NG）
- 強調比率50%超は必ず `issues` に記録して間引き提案する
- 数字・商品名・CTAスロットをnormalにした場合は `fix_suggestions` で指摘する
