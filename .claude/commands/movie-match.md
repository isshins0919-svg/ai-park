# 🧩 動画マッチクルー ver.1.0 — 「最適な積荷を最適な場所に」

30スロット全部に「最も説得力が上がるクリップ」を割り当てる。
テキストと映像がずれている動画は、どんなにコピーが良くても刺さらない。

> **動画の基本定義**: 60秒 = 30スロット × 2秒固定グリッド
> マッチクルーは全30スロットに clip_ref を割り当てる。

`/movie-match` で起動。スロットのジョブ+テキスト（動画テンポクルー出力）+ 利用可能なクリップ一覧を渡す。

## マッチングの優先順位

```
1. テキストの内容と映像が一致している（最優先）
   例: 「皮脂が増える」→ 顕微鏡・肌アップ映像
   例: 「1日1杯で」→ 商品を飲んでいるシーン

2. ジョブと映像のトーンが合っている
   フック → インパクト強いビジュアル
   問題深掘り → 不安感を煽る映像（暗め・アップ）
   解決策 → 明るい・商品映像
   実績 → 笑顔・ビフォーアフター
   CTA → 商品正面・価格テロップ重ね

3. 同じクリップを3スロット以上連続させない
```

## ジョブ別 推奨映像タイプ

| ジョブ | 推奨映像タイプ |
|---|---|
| ⚡ フック | 顔アップ・衝撃的な映像・問題の視覚化 |
| 😟 問題提起 | 不安顔・問題状況・before映像 |
| 🤝 共感 | 共感顔・日常シーン・うなずき |
| 🔥 問題深掘り | 問題の拡大・数字テキスト・衝撃映像 |
| 💡 解決策登場 | 商品登場・手に持つ・パッケージ |
| 📖 解決策説明 | 成分説明・図解・商品アップ |
| ✅ 証拠・実績 | 笑顔・コメント表示・数字テロップ |
| 🚀 期待・高揚 | 明るい笑顔・before→after・ライフスタイル |
| 🎯 CTA前振り | 商品＋人物・期待顔 |
| 📣 CTA | 商品正面アップ・価格テロップ・購入ボタン |

## 制約条件

- **やること**: 全30スロットへのclip_ref割り当て + マッチ強度評価
- **やらないこと**: テキスト修正・感情強度変更・尺の変更
- 全30スロットを出力する（省略NG）
- confidence 0.6未満のスロットは必ず `weak_matches` に記録し代替案を出す
- 同じクリップが3スロット以上連続する場合は `same_clip_consecutive_warning` に記録する
- クリップが存在しない場合は `clip_ref: null` と記録し、撮影または生成が必要と伝える
