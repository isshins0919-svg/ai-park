# Amazon Park ver.1.0 — Amazon出店 × 商品ページ × 画像生成 一気通貫スキル

## 目的
「Amazon出店したい」から「商品ページ完成 + 画像 + 広告初期戦略」まで、
クライアント名と商品情報を渡すだけで一気通貫でアウトプットする。

---

## 発動トリガー

- 「Amazon出店スキル」
- 「〇〇のAmazon出店進めて」
- 「〇〇のAmazonページ作って」
- 「amazon park」
- 「Amazon商品ページ作って」

---

## 実行フロー

```
PHASE 1: 商品情報収集（ヒアリング or クライアントファイル読み込み）
    ↓
PHASE 2: 出店チェックリスト確認
    ↓
PHASE 3: 商品ページコピー生成（タイトル / 箇条書き / 商品説明）
    ↓
PHASE 4: 画像構成設計（Nano Banana Pro用プロンプト）
    ↓
PHASE 5: 広告初期戦略（PPC / キーワード）
```

---

## PHASE 1: 商品情報収集

クライアントファイルがあれば自動読み込み。なければ以下をヒアリング:

```
Amazon出店に必要な情報を教えて！

【商品基本情報】
1. 商品名（正式名称）
2. カテゴリ（例: 靴下 / スキンケア / サプリ）
3. ターゲット（誰に売る？）
4. 価格帯（想定）
5. 商品の一番の強み（1つだけ）
6. 競合と比べて何が違う？

【素材・仕様】
7. 素材・成分・スペック（箇条書きでOK）
8. サイズ・数量・入数

【画像素材】
9. 実物の写真はある？（ある / ない / これから撮る）
10. ブランドカラー・雰囲気のイメージ
```

## PHASE 2: Amazon出店チェックリスト

### セラーアカウント
- [ ] Amazonセラーセントラルのアカウント開設（個人 or 法人）
- [ ] 銀行口座・クレジットカード登録
- [ ] 本人確認書類の提出（法人は登記簿謄本）
- [ ] 出品プラン選択（小口 / 大口 — 月49点以上なら大口推奨）

### 商品登録に必要なもの
- [ ] JANコード（バーコード）— 取得が必要な場合はGS1申請
- [ ] 商品の主カテゴリ・サブカテゴリの確認
- [ ] ブランド登録（Amazon Brand Registry — 商標があれば推奨）
- [ ] FBA利用有無の決定（FBA = Amazon倉庫に預ける）

### コンテンツ
- [ ] 商品タイトル（最大200文字）
- [ ] 商品説明・箇条書き（5項目）
- [ ] 商品説明文（A+コンテンツ — ブランド登録後）
- [ ] メイン画像（白背景 / 長辺1000px以上）
- [ ] サブ画像（最大8枚）
- [ ] 価格設定

### 法規制確認（カテゴリ別）
- [ ] 食品・サプリ → 食品表示法・薬機法チェック
- [ ] アパレル → 家庭用品品質表示法チェック
- [ ] 輸入品 → 関税・PSE・PSCマーク確認

## PHASE 3: 商品ページコピー生成

### タイトル設計（Amazon SEO最適化）

**構成テンプレ**:
```
[ブランド名] [商品名] [一番の特徴] [ターゲット/シーン] [スペック]
```

**例（靴下の場合）**:
```
[CAMICKS] 靴下 メンズ 蒸れない 抗菌防臭 ビジネスソックス 5足セット 25-27cm
```

**タイトルのルール**:
- 重要キーワードを前半に入れる
- 数字・スペックを入れると検索に強い
- 禁止: 「最高」「No.1」「最安値」等の誇大表現
- 文字数: 80〜120文字が最適

---

### 商品説明（箇条書き5項目）設計

**構成テンプレ**:
```
① 一番の強み（ベネフィット）
② 素材・品質の根拠
③ 使用シーン・ターゲット
④ スペック・仕様
⑤ アフターサービス・保証
```

各項目: 150〜200文字。先頭に絵文字か【】でキーワードを入れると読まれやすい。

---

### 商品説明文（本文）設計

**構成**:
```
1. 共感（こんなお悩みありませんか？）
2. 解決策（この商品が解決します）
3. 根拠（素材・製法・数字）
4. 使用シーン
5. ブランドの想い
```

## PHASE 4: 画像構成設計（Nano Banana Pro）

### Amazon必須画像セット（最大9枚）

| 枚数 | 内容 | 役割 |
|-----|------|------|
| 1枚目（メイン） | 白背景 + 商品のみ | 検索結果に表示される最重要画像 |
| 2枚目 | ライフスタイル（着用・使用シーン） | 「自分ごと」イメージ |
| 3枚目 | 素材・品質クローズアップ | 品質の証明 |
| 4枚目 | 強み比較表（競合比較） | 選ばれる理由 |
| 5枚目 | スペック・サイズ詳細 | 購入判断サポート |
| 6枚目 | 使い方・シーン展開 | 活用イメージ拡張 |
| 7枚目 | ブランドストーリー / 製造こだわり | ブランド信頼 |
| 8枚目〜 | レビュー引用 / Q&A / セット展開 | 不安解消 |

### Nano Banana Pro 画像生成プロンプトテンプレ

```
【メイン画像用プロンプト】
商品: [商品名]
背景: 純白 (#FFFFFF)
ライティング: スタジオ照明、影少なめ
アングル: 正面 or 斜め45度
品質: Amazon商品撮影規格、プロ仕上げ、高解像度
スタイル: ミニマル、清潔感

【ライフスタイル画像用プロンプト】
シーン: [使用シーン]
モデル: [ターゲット像]
雰囲気: [ブランドトーン]
商品の見え方: 自然な使用状態
色調: [ブランドカラー]
```

### Camicksで確立済みのワークフロー
- Nano Banana Pro（gemini-3-pro-image-preview）で生成
- 関連ファイル: `image-gen/` 配下のCamicksプロセスを参照
- 同じフローで他商品にも転用可能

## PHASE 5: 広告初期戦略（PPC）

### 出品直後の戦略（〜1ヶ月）

**目的**: レビュー0件でも露出を稼ぎ、最初の売上・レビューを獲得する

**推奨キャンペーン構成**:
```
① オートターゲティング（まず走らせる）
   → Amazonが自動でキーワードを探してくれる
   → 予算: 500〜1,000円/日
   → 2週間データを取ってから手動に移行

② スポンサープロダクト（手動・完全一致）
   → 主要キーワード3〜5個に絞る
   → 入札単価: カテゴリ相場の1.2〜1.5倍でスタート
```

**キーワード選定の考え方**:
- メインキーワード（商品名 + カテゴリ）
- 競合ブランド名（出せる場合）
- 用途・シーンキーワード（「通勤 靴下」「ビジネス ソックス」）

**目標数値（初月）**:
- ACOS（広告費用対売上）: 50%以下を目指す
- レビュー: 5件獲得を最優先

---

## 出店後のネクストアクション

| タイミング | アクション |
|----------|-----------|
| 出品直後 | オートキャンペーン開始 + レビュー依頼設定 |
| 2週間後 | 広告データ分析 → 手動キャンペーンに移行 |
| 1ヶ月後 | レビュー5件達成 → A+コンテンツ追加 |
| 3ヶ月後 | ランキング確認 → 価格・キーワード最適化 |
