長文禁止。スッキリ読める形で、チームの朝を情報武装する。

## 実行フロー（毎朝8:00）

### Phase 1: 媒体ID取得
```
read_apps で Instagram / YouTube の app_id を取得
（毎回確認。IDは変わる可能性があるため決め打ちNG）
```

### Phase 2: ジャンルID取得
```
search_genres で以下のキーワードを検索:
- 「毛穴」「シミ」「シワ」「美肌」
- 「ヒザ」「膝」「関節」
- 「成長」「子供」「サプリ」「身長」
- 「体臭」「デオドラント」「ワキガ」
各ジャンルのIDを取得してから items_get に渡す
```

### Phase 3: データ取得（5ジャンル × 2媒体 = 最大10クエリ）
```
items_get のパラメータ:
- genre_id: 上記で取得したID
- app_id: Instagram / YouTube
- sort: 推定広告費増加額（直近30日）降順
- limit: 3件
```

## 投稿ルール（厳守）

- **チャンネルへの投稿は1回だけ**（メインサマリーのみ）
- **詳細はすべてスレッド内へ**（チャンネルを汚さない）
- **1スレッド投稿 = 1ジャンル × 1媒体**
- 数字は「推定値」として扱う
- 商品の実名は出してOK（DProのデータに基づく）
- エラー時は「⚠️ 本日のデータ取得に失敗しました」とだけ投稿して終了
